### 2.0.1 Do the logarithm of the expression data

Why can’t we use raw data for PCA right away? For PCA, we need
logarithmically smoothed data; outliers in the data will greatly distort
the picture. Expression differences can vary by a factor of a million.
We need to reduce the absolute difference between features; PCA works
with absolute variances, so they need to be strictly logarithmized.
Conventionally, we need to narrow the broad scale.

If the difference is three orders of magnitude, then 10 log; if less
than 2 log. But if the range is small, then, conversely, we take the
square root. (The larger the logarithm base, the smaller the range the
data will be in)

What order of difference? (6 - 10,000, for example) =\> log10.

``` r
log_data <- lapply(norm_data, log10)
```

# 6 Differential expression

First, we’ll calculate differential expression using a t-test and the
nonparametric Mann-Whitney test. Limma is the core of edgeR, DESEQ, and
other tools.

> Limma is a general-purpose package that can be used in any analysis on
> any data, as it doesn’t have heuristics specific to specific data
> types and applications, like deseq for ngs. Limma builds a negative
> binomial distribution model and adjusts the weights of different genes
> to fit it. It works well with small samples, but for samples larger
> than 500, classic methods (t-test, U-test) can be used.

``` r
# no outliers
norm_data_cleaned <- norm_data[!(colnames(norm_data) %in% c('C_209', 'C_210', 'C_211', 'Dfu_209', 'Dfu_210', 'Dfu_223'))]
meta4limma_test <- meta4limma[!(meta4limma$Sample_name %in% c('C_209', 'C_210', 'C_211', 'Dfu_209',  'Dfu_210', 'Dfu_223')), ]
```

## 6.1 t-test, u-test, limma

What’s the test’s direction – we divide DFU by Control.

- LFC \> 0 → up‑regulation in FECD.

- LFC \< 0 → down‑regulation.

We need to remove the miRNA where **BOTH values ​​are low across groups**,
as such pairs **are artifactual**. A value where one is 1 and the other
is 0 is ok. We’ll immediately filter them out of the DEGs.

``` r
gr_1 <- sum(grepl("C_", colnames(norm_data_cleaned)))   #8 end of 1 gr
gr_2 <- gr_1 + 1 #  2 gr start


deg_ttest <- do.call(rbind, 
                      lapply(seq(1, nrow(norm_data_cleaned)), function(x)
  {f_tt <- t.test(norm_data_cleaned[x, 1:gr_1], norm_data_cleaned[x, gr_2:ncol(norm_data_cleaned)])
  
  mean_c <- (mean(unlist(norm_data_cleaned[x, 1:gr_1])))
  mean_f <- (mean(unlist((norm_data_cleaned[x, gr_2:ncol(norm_data_cleaned)]))))
  
  fc <- log2(mean_f / mean_c)
  
  return (unlist(c(fc, f_tt$p.value, mean_f, mean_c)))
  }
))

deg_ttest <- as.data.frame(deg_ttest)
colnames(deg_ttest) <- c('lfc', 't_pval', 'mean_f', 'mean_c')
rownames(deg_ttest) <-  other_data$`Probe Name`
deg_ttest <- deg_ttest %>% rownames_to_column('miRNA')

# filter ZERO miRNA in BOTH groups
low_mirna <- deg_ttest %>% filter((mean_c < 2.2) & (mean_f < 2.2)) %>% pull(miRNA)
deg_ttest <- deg_ttest %>% filter(!(miRNA %in% low_mirna))
deg_ttest$p_adj <- p.adjust(deg_ttest$t_pval, "fdr")

# u-test
deg_utest <- do.call(rbind, 
                      lapply(seq(1, nrow(norm_data_cleaned)), function(x)
  {u_tt <- wilcox.test(as.numeric(norm_data_cleaned[x, 1:gr_1]), as.numeric(norm_data_cleaned[x, gr_2:ncol(norm_data_cleaned)]))
  fc <- log2(
    mean(unlist((norm_data_cleaned[x, gr_2:ncol(norm_data_cleaned)]))) / mean(unlist(norm_data_cleaned[x, 1:gr_1]))
    )
  return (unlist(c(fc, u_tt$p.value)))
  }
))
deg_utest <- as.data.frame(deg_utest)
colnames(deg_utest) <- c('lfc', 'u_pval')
deg_utest$p_adj <- p.adjust(deg_utest$u_pval, "fdr")
rownames(deg_utest) <-  other_data$`Probe Name`
deg_utest <- deg_utest %>% rownames_to_column('miRNA') %>% filter(!(miRNA %in% low_mirna))

#limma
# create groups based on metadata
group <- factor(meta4limma_test$FECD, levels = c("no", "yes"))
# we create a design matrix, ~0 means without including the intercept (0)   
design <- model.matrix(~0 + group)
colnames(design) <- c("Control", "FECD")
# into matrix and log
limma_matrix <- as.matrix(norm_data_cleaned)
limma_matrix <- log2(limma_matrix)

# applying a linear regression model to a matrix
limma_fit <- lmFit(limma_matrix, design)

# Contrasts for comparing groups
contrast_matrix <- makeContrasts(FECD - Control, levels = design)
fit2 <- contrasts.fit(limma_fit, contrast_matrix)
# Bayesian analysis
fit2 <- eBayes(fit2)

# retrieving results
deg_limma <- topTable(fit2, adjust.method = "fdr", number = Inf)
deg_limma <- deg_limma %>% mutate(miRNA = rownames(.)) %>% filter(!(miRNA %in% low_mirna))
rm(fit2, limma_fit, limma_matrix, design, group, meta4limma_test)
```
